# Clutch Bot 🎮

A Discord bot built for gamers. Clutch keeps everything inside your server so you never have to open a new tab.

## Features

- 🆓 `/freegames` — Live free games on Epic Games Store
- 🔥 `/deals` — Hottest game deals across Steam, Epic, GOG and more
- 📅 `/upcoming` — Upcoming game releases
- 🔍 `/game` — Full game info, ratings, cover art and description
- 🎯 `/suggest` — Game suggestions by genre or random
- 🎬 `/walkthrough` — YouTube walkthroughs for any game or mission
- ❓ `/help` — Show all commands
- 🏓 `/ping` — Check if bot is online

## Built With

- Node.js
- discord.js
- RAWG API
- CheapShark API
- Epic Games API
